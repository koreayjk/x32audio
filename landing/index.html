음향 전문가가 없어도
괜찮습니다

이제는 누구나 음향 믹서를 전문가처럼 잡을 수 있습니다.

복잡한 콘솔 대신 "설교 / 찬양팀 / 기도" 버튼 하나로 전환하고, 본당·방송·모니터 세 출력을 동시에 관리하며, 하울링은 실시간으로 잡아냅니다. 음향이 처음인 봉사자도 바로 쓸 수 있는 교회 음향 자동화 데스크탑 앱.

설치 후 X32의 IP만 입력하면 바로 연결됩니다.

주요 기능

현장에 필요한 것만, 쉽게

매주 바뀌는 음향팀 누구나 5분이면 익히도록 만들었습니다.

🎬

원터치 Scene · 큐

설교·찬양팀·기도·광고·전체음소거를 버튼 하나로. 예배 순서를 큐로 묶어 스페이스바로 차례차례 전환합니다.

바로 사용
🔊

3개 출력 동시 관리

본당(Main LR)·방송(Bus 1)·찬양팀 모니터(Bus 2)를 각각 다른 믹스로. 예배 시작 버튼 하나로 세 출력을 동시에 세팅합니다.

바로 사용
📺

방송 자동 레벨 · 설교 모드

방송 버스를 유튜브 기준 LUFS −14로 자동 보정. 설교 때는 방송에서만 악기를 자동으로 낮춰줍니다.

바로 사용
🛡️

피드백 감지 + 자동 억제

하울링(삐— 소리) 주파수를 실시간 감지·경고하고, GEQ에서 자동으로 눌러줍니다. 민감도 조절 가능.

바로 사용
🎚️

양방향 페이더 제어

앱에서 페이더를 끌면 콘솔이 움직이고, 콘솔에서 바꾸면 앱에 실시간 반영. 채널 상태·4밴드 EQ도 한눈에.

바로 사용
🎙️

인물별 마이크 프리셋

"담임목사님"처럼 사람별 EQ·로우컷을 저장해, 그날 쓰는 채널에 그대로 적용합니다.

바로 사용
📱

태블릿·폰 원격 조작

같은 와이파이의 기기 브라우저에서 Scene 전환·예배 순서 큐·전체 음소거를 원격으로. 별도 설치 불필요.

바로 사용
💾

저장 · 되돌리기 · 백업

사용자 정의 Scene 저장, 직전 상태로 되돌리기(Undo), 설정·콘솔 상태를 파일로 백업/복원.

바로 사용
🧩

봉사자 모드 · 다국어

큰 버튼 위주의 쉬운 모드와 한국어/English 전환. 첫 실행 단계별 가이드까지.

바로 사용
사용법

처음이어도 괜찮아요

앱을 처음 켜면 단계별 가이드가 화면을 짚어주며 따라오게 안내합니다.

1

연결

X32의 IP 주소를 입력하고 [연결]. 콘솔 정보가 자동으로 표시됩니다.

2

상태 확인

[상태 읽기]로 현재 채널 볼륨·음소거·EQ를 한눈에 불러옵니다.

3

장면 전환

예배 순서대로 Scene 버튼을 누르거나, 큐로 묶어 스페이스바로 진행.

4

안심 모니터

피드백 감지를 켜두면 하울링을 실시간 경고하고 자동으로 눌러줍니다.

차별점

하울링, 감지에서 끝내지 않습니다

"삐—" 소리가 예배를 방해하기 전에, 우는 주파수를 찾아 자동으로 눌러줍니다.

  • RTA 스펙트럼에서 지속되는 좁은 피크를 실시간 감지
  • 해당 주파수를 31밴드 GEQ의 가장 가까운 밴드로 매핑
  • 단계적으로 감쇠하고, 안전 한도를 넘지 않도록 제한
  • 끄면 깎았던 밴드를 0dB로 자동 복원
⚠ 2.0kHz 피드백 감지 → 자동 −6dB
지원 믹서
