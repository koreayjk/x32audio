복잡한 콘솔 대신 "설교 / 찬양팀 / 기도" 버튼 하나로 전환하고, 하울링은 실시간으로 잡아냅니다. 음향이 처음인 봉사자도 바로 쓸 수 있는 교회 음향 자동화 데스크탑 앱.

주요 기능

현장에 필요한 것만, 쉽게

매주 바뀌는 음향팀 누구나 5분이면 익히도록 만들었습니다.

🎬

원터치 Scene

설교·찬양팀·기도·광고·전체음소거를 버튼 하나로 일괄 적용. 채널 음소거와 볼륨이 한 번에 세팅됩니다.

바로 사용
📋

예배 순서 큐

예배 순서를 미리 짜두고 스페이스바로 다음 장면을 차례차례 전환. 진행자도 흐름을 놓치지 않습니다.

바로 사용
⚠️

피드백 감지·경고

RTA 분석으로 하울링(삐— 소리) 의심 주파수를 실시간 경고. 민감도도 조절할 수 있어요.

바로 사용
🛡️

자동 피드백 억제

감지에서 끝나지 않습니다. 우는 주파수를 GEQ에서 자동으로 단계적 감쇠해 하울링을 눌러줍니다.

바로 사용
💾

사용자 정의 Scene

지금 콘솔 상태를 "주일예배 1부"처럼 이름 붙여 저장하고, 다음 주에 그대로 불러오기.

바로 사용
📊

채널 상태 한눈에

채널별 이름·볼륨(dB)·음소거·EQ 상태를 표로 확인. 4밴드 EQ 상세까지 한 화면에.

바로 사용
사용법

처음이어도 괜찮아요

앱을 처음 켜면 단계별 가이드가 화면을 짚어주며 따라오게 안내합니다.

1

연결

X32의 IP 주소를 입력하고 [연결]. 콘솔 정보가 자동으로 표시됩니다.

2

상태 확인

[상태 읽기]로 현재 채널 볼륨·음소거·EQ를 한눈에 불러옵니다.

3

장면 전환

예배 순서대로 Scene 버튼을 누르거나, 큐로 묶어 스페이스바로 진행.

4

안심 모니터

피드백 감지를 켜두면 하울링을 실시간 경고하고 자동으로 눌러줍니다.

차별점

하울링, 감지에서 끝내지 않습니다

"삐—" 소리가 예배를 방해하기 전에, 우는 주파수를 찾아 자동으로 눌러줍니다.

  • RTA 스펙트럼에서 지속되는 좁은 피크를 실시간 감지
  • 해당 주파수를 31밴드 GEQ의 가장 가까운 밴드로 매핑
  • 단계적으로 감쇠하고, 안전 한도를 넘지 않도록 제한
  • 끄면 깎았던 밴드를 0dB로 자동 복원
⚠ 2.0kHz 피드백 감지 → 자동 −6dB
